# Project Overview

## Purpose

The Earthquake Notification System (ENS) is a real-time platform designed to collect, process, and distribute earthquake information to mobile users on Android and iOS devices.

## Objectives

- Provide near real-time earthquake notifications.
- Integrate with SeisComP seismic processing systems.
- Deliver reliable and scalable services.
- Support thousands of concurrent mobile users.
- Provide monitoring, logging, and disaster recovery mechanisms.

## Main Components

- Mobile Applications
- Backend API
- Notification Service
- SeisComP Integration Service
- Database
- Monitoring Infrastructure

## Target Users

- General public
- Researchers
- Government agencies
- Emergency response organizations
